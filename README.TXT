Titulo del proyecto: Ejercicio 3-Resta en un String
Proposito del proyecto: Realizar una resta de dos numeros introducidos en una misma linea
Version o fecha: 19/04/2020
Como iniciar este proyecto: Ejecutar el main de la clase
Autores: Franco Matias Abiu Mahanaim 19161268
Instrucciones para el usuario:
Para prueba del programa se debe de ejecutar el main de la clase y ingresar los dos numeros separados
por un solo espacio.
Solucion: En este programa lo primero que se tuvo que sacar fue un String del inicio hasta donde 
se encontrara el espacio, y hacer lo mismo desde el espacio hasta el ultimo digito, despues se sacaron los
caracteres de cada uno de los strings para despues sumarlos y por ultimo convertirlos a enteros y restarlos
